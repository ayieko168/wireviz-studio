WireViz Studio

Desktop YAML-to-diagram workflow for wiring harnesses, with BOM preview and one-click exports.

Download Releases Documentation Syntax Reference

Build Targets

  • Windows installer (.exe)
  • Windows portable (.zip)
  • macOS disk image (.dmg)
  • Linux AppImage (.AppImage)

Core Features

  • Multi-tab YAML editor with highlighting
  • Manual render for predictable iteration
  • Diagram and BOM tabbed preview
  • PNG, SVG, PDF, and CSV export
